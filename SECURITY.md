# Security Policy

The Lector project prioritizes security and strives to maintain a safe environment for all users. We encourage contributors and users to examine the codebase for potential vulnerabilities and to share any concerns they find. Suggestions for features that enhance security and meet user needs are also welcome.

## Reporting a Vulnerability

If you identify a critical vulnerability in this project, please report it responsibly through GitHub's private [security advisories](https://github.com/lector-org/lector/security/advisories).
